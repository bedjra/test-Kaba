Étant donné un tableau numsde taille n, renvoyez l'élément majoritaire .

L'élément majoritaire est l'élément qui apparaît plus de ⌊n / 2⌋fois. On peut supposer que l'élément majoritaire existe toujours dans le tableau.

 

Exemple 1 :

Entrée : nums = [3,2,3]
 Sortie : 3
 
Exemple 2 :

Entrée : nums = [2,2,1,1,1,2,2]
